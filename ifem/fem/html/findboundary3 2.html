function [bdNode,bdFace] = findboundary3(elem)

FINDBOUNDARY3 find boundary faces and nodes

[bdNode,bdFace] = findboundary3(elem) find the boundary faces and nodes
of a 3-dimensional tetrahderon mesh. Note only topological structure of
the mesh is used.
 See also findboundary, showboundary3.

allFace = [elem(:,[2 4 3]);elem(:,[1 3 4]);elem(:,[1 4 2]);elem(:,[1 2 3])];
[face, i2, j] = unique(sort(allFace,2),'rows');
NT = size(elem,1);
i1(j(4*NT:-1:1)) = 4*NT:-1:1; i1 = i1';
bdFace = face((i1 == i2),:);
isBdNode(bdFace) = true;
bdNode = find(isBdNode);
